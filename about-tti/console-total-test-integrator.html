Total Test Integrator - Консольная версия

Дополнительно к версии с GUI интерфейсом можно использовать консольную версию ttie.exe. Она так же как и GUI версия собрана для Windows, однако может выполняться только в текстовом режиме коммандной строки. Консольная версия разработана в Qt и может быть собрана для различных операционных систем.

По сути консольная версия предназначена для выполнения планов тестирования, подготовленные в GUI версии или вручную. Результаты тестирования так же сохраняются в базу данных, которая может быть загружена для анализа в GUI. Консольная версия может быть использована при запуске из под планировщика заданий или по событию в системах Continuos Integration.

Формат запуска консольной версии:

tti.exe [[путь к плану тестирования] [комментарий или путь к файлу содержащему комментарий]] ...
