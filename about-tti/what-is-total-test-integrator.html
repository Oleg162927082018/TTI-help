Total Test Integrator - Общие сведения

Total Test Integrator (TTI) - это приложение имеющее целью объединить различные типы тестов в единую систему запуска, отображения и анализа. TTI мощный инструмент позволяющий связать результаты тестирования с различными системами багтрекинга, либо системами хранения информации, проводить регрессионный анализ или отслеживать работу над заданием.

Приложение разработано в системе Qt. Это означает, что оно может быть собрано для различных операционных систем, в частности, для Windows, Linux, Mac OS X, а также и для мобильных платформ.


Total Test Integrator для Windows поставляется в двух вариантах:

  • версия с графическим интерфейсом пользователя (GUI) - tti.exe;
  • консольная версия - ttie.exe, запускаемая из командной строки и позволяющая запустить тесты по времени или по событию в фоновом режиме.
